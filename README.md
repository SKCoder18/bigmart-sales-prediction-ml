# 🛒 Big Mart Sales Prediction using Machine Learning

## 📌 Project Overview
This project predicts the sales of products in Big Mart outlets using Machine Learning.  
It helps businesses make better decisions on inventory and sales forecasting.

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 📊 Sales Prediction using ML Model
- 💾 Save Predictions to CSV
- 📦 View Product Data
- 🌐 Full Stack Web Application (Frontend + Backend)

## 🧠 Machine Learning
- Algorithms Used:
  - Linear Regression
  - Decision Tree
  - Random Forest (Final Model)
  - XGBoost
- Final Model: Random Forest Regressor

## ⚙️ Tech Stack
- Python
- Flask (Backend)
- HTML, CSS, JavaScript (Frontend)
- SQLite (Database)
- Pandas, NumPy, Scikit-learn

## 📂 Dataset
- Source: Kaggle Big Mart Sales Dataset

## 🔄 Workflow
1. Data Cleaning & Preprocessing
2. Feature Engineering
3. Model Training
4. Model Evaluation
5. Deployment using Flask API
