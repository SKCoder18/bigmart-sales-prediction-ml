About

Big-Mart Sales Predictor is a machine learning-based web application that predicts future sales based on historical data.

It helps businesses optimize their inventory, and make informed decisions.

Work Done

We've took the dataset named Big Mart Sale Data dataset from Kaggle open source repository and we preprocessed it.

Preprocessing:

By using Python we followed the following steps to clean and preprocess the data Load the data, Handling missing values, Handling duplicate data, Handling outliers, Encoding Categorical Variables, Final Check on Cleaned Data.

After that the dataset is divided into test and train datasets.

Model Selection:

Then we've selected the models to train. They are Linear regression, Decision Tree, Random Forest, XGboost.

Model Training:

Final Selected Model:

After analyzing the evaluation metrics and graphs we selected the best model i.e Random Forest
